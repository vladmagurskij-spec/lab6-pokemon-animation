Анімація спрайта Покемона (Генгар)

Мета: попрацювати зі спрайтами, анімацією сцени та чистим CSS.

Обов’язково реалізовано:

  • Анімація спрайта Генгара за допомогою @keyframes + background-position (sprite sheet).
  • Анімація фону (зоряне небо, туман, паралакс пагорбів).
  • Чистий CSS для всієї анімації (JS лише для кнопки паузи).
